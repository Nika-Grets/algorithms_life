# Учебные проекты

Коллекция проектов, демонстрирующая навыки работы с алгоритмами и чистым JavaScript.

### Включенные проекты:
1. Conway's Game of Life:
   * Реализация на Canvas API.
   * Оптимизированный цикл пересчета поколений.
   * Возможность настройки скорости и размера поля «на лету».
2. Алгоритмический сервер (Node.js/Fastify):
   * Реализация рекурсивных вычислений (факториалы, числа Фибоначчи).
   * Рекурсивный обход сложных вложенных структур данных (расчет фонда оплаты труда).
   * Использование шаблонизатора EJS для рендеринга результатов.

### Ключевые навыки:
Алгоритмическая сложность, рекурсия, работа с DOM, Node.js Backend.
